# Obsidian config

> Obsidian config I use for my vaults

## Usage

Copy the `.obsidian` to your vault, adapt configs as needed and have fun.

## Features

1. Vim mode and config
2. Mind map and table editor support
3. Custom keybindings and config
4. And muuch more
